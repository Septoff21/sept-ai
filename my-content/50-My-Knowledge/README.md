# My Knowledge

Hand-written notes — the "me" layer of Sept-AI. Linked from `/me` on the site.

## Sub-folders

- **[[Claude-Learning]]** — synced from [Septoff21/claude-learning](https://github.com/Septoff21/claude-learning) (weekly, via GitHub Actions)
- **[[Prompts]]** — reusable prompt templates, one per file
- **[[Workflows]]** — multi-step patterns (e.g. "Claude + Cursor for refactoring")

Anything here is fair game to edit, rename, reorganize. Use Obsidian's Graph view to see how notes connect.

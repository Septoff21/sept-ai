v0.5.0 发布说明

主要更新:
- Provider CLI 管理：agencycli provider add/list/remove/set-default
- 首次使用引导向导（setup guidance）
- Web UI 内联 Cron 编辑（表达式/提示词/开关）
- 心跳休眠阶段即时触发到期 Cron
- Agent 身份环境变量自动注入
- 终止运行中 Agent（Web + CLI）
- Sandbox 配置面板（镜像/挂载/环境变量）
- 可拖拽排序的页面标签栏
- 多项 Docker 沙箱修复（PATH展开、stdin管道、读写挂载）

安装/升级:
npm install -g @agencycli/agencycli
